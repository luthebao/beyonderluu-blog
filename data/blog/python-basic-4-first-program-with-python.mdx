Hello mọi người.

Ở bài viết trước mình đã hướng dẫn cài đặt môi trường python lên máy tính và công cụ editor để thuận tiện cho việc lập trình python. Và bây giờ chúng ta hãy cùng học cách lập trình python nhé.

z/blog/pythonbasic/1/1.jpg" width="300" height="150" />

Đầu tiên, chúng ta hãy mở Visual Studio Code lên để tạo 1 folder riêng cho việc học python của mình.

Ví dụ mình tạo 1 folder tên là python để ở ngoài màn hình (Desktop). Sau đấy quay lại VSCode để vào folder đó bằng cách ấn vào "Open folder..."

<Image src="/blog/pythonbasic/4/1.png" width="300" height="150" />

Sau khi mở được folder python vừa tạo, nó sẽ hiện lên như thế này

<Image src="/blog/pythonbasic/4/2.png" width="300" height="150" />

Tiếp theo chuột phải bên phần folder Python chọn New file, để tạo tập tin cho chương trình python. Đặt tên cho nó là "test1.py". Các chương trình python thường sẽ có đuôi là ".py" nhé.

<Image src="/blog/pythonbasic/4/3.png" width="300" height="150" />

Lúc tạo file ".py" xong, VSCode sẽ hiện gợi ý cài đặt 1 cái tiện ích nào đó, các bạn ấn vào Install để cài đặt cho thuận lợi việc code python nhé.

<Image src="/blog/pythonbasic/4/4.png" width="300" height="150" />

Bây giờ chúng ta gõ câu lệnh đầu tiền nhé. 

Bài học vỡ lòng: In ra màn hình bằng câu lệnh print(X)

Trong đó: X có thể là giá trị số hoặc chữ. Nếu là chữ (gọi là kiểu string) thì sẽ để trong dấu nháy đơn hoặc nháy kép. Ví dụ:

```py
print(1) 
print("Chương trình đầu tiên")
```

Sau khi gõ xong, các bạn có thể ấn vào hình tam giác màu xanh của VSCode để chạy, kết quả của chương trình sẽ hiển thị ở phía bên dưới phần terminal của VSCode.

<Image src="/blog/pythonbasic/4/5.png" width="300" height="150" />

Vậy là xong chương trình đầu tiên của các bạn rồi phải không nào, hãy đón chờ các bài viết tiếp theo để nâng cao thêm 1 tí nhé 💪
